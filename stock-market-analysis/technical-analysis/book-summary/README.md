# Technical Analysis — Book Summaries

Summaries and notes from prominent **technical analysis** books. Use these alongside the [Technical Analysis Handbook](../handbook/README.md) for deeper theory and real-world application.

---

## Recommended Reading Order

For beginners, a suggested order is:

1. **Getting Started in Technical Analysis** — Foundations and chart types.
2. **Technical Analysis Explained** — Indicators and systems.
3. **Japanese Candlestick Charting Techniques** — Candlestick patterns in depth.
4. **Encyclopedia of Chart Patterns** — Chart patterns reference.
5. **Technical Analysis Using Multiple Timeframes** — Multi-timeframe approach.
6. **Technical Analysis of the Futures Markets** / **How to Make Money in Stocks** — Markets and application.
