# Stock market analysis

This folder contains the main **three pillars** of the TradingOverview repository: **Technical Analysis**, **Fundamental Analysis**, and **Options Trading**. Each pillar has its own README, book notes or summaries, and (where applicable) a full handbook.

---

## The three pillars

| Pillar | What it covers | Main entry |
|--------|----------------|------------|
| **Technical Analysis** | Price action, patterns, indicators, strategies, risk, psychology, backtesting | [technical-analysis/README.md](technical-analysis/README.md) |
| **Fundamental Analysis** | Financial statements, ratios, valuation, qualitative factors, value vs growth | [fundamental-analysis/README.md](fundamental-analysis/README.md) |
| **Options Trading** | Calls/puts, Greeks, strategies, IV, risk, hedging | [options-trading/README.md](options-trading/README.md) |

---

## How they work together

- **Fundamental Analysis** → *What* to buy or sell (quality, value, growth).
- **Technical Analysis** → *When* to enter/exit and *where* to set stops and targets.
- **Options Trading** → *How* to structure the exposure (leverage, income, hedging).

For a unified overview and one-page summaries, use the **[Summary folder](../Summary/README.md)** at the repo root. For practical tips, use the **[Tips_and_Tricks folder](../Tips_and_Tricks/README.md)**.
