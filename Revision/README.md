# Revision — Quick Reference & Cheat Sheets

This folder is for **exam-style revision** and **quick lookup** of the Technical Analysis & Trading Handbook. Use it when you need to recall definitions, patterns, formulas, or checklists without re-reading the full handbook.

---

## Contents

| Document | Use |
|----------|-----|
| **[Complete_Trading_Revision.md](Complete_Trading_Revision.md)** | **Start here.** One-doc summary of the entire handbook: market basics, price action, candlesticks, chart patterns, indicators, advanced concepts, strategies, risk, psychology, backtesting, algo, common mistakes. Key definitions and ASCII charts. |
| **[Trading_Patterns_Cheat_Sheet.md](Trading_Patterns_Cheat_Sheet.md)** | All candlestick and chart patterns: recognition rules, bias, stop placement. Reversal and continuation patterns with measured moves. |
| **[Indicators_Quick_Reference.md](Indicators_Quick_Reference.md)** | Trend, momentum, volatility, and volume indicators: what they measure, main signals, best market conditions. |
| **[Trading_Setups_Checklist.md](Trading_Setups_Checklist.md)** | Pre-trade, execution, and post-trade checklists. Weekly and monthly review prompts. |
| **[Risk_Management_Formula_Sheet.md](Risk_Management_Formula_Sheet.md)** | Position sizing formulas, dollar risk, R:R, expectancy, Kelly. Example calculations for stocks, forex, futures. |

## When to Use

- **Before trading:** Run through [Trading_Setups_Checklist.md](Trading_Setups_Checklist.md) and [Risk_Management_Formula_Sheet.md](Risk_Management_Formula_Sheet.md).
- **Pattern or indicator lookup:** Use [Trading_Patterns_Cheat_Sheet.md](Trading_Patterns_Cheat_Sheet.md) and [Indicators_Quick_Reference.md](Indicators_Quick_Reference.md).
- **Full revision:** Read [Complete_Trading_Revision.md](Complete_Trading_Revision.md) end-to-end for a refresher of the whole handbook.
